#  ChocoRush Sales Analysis Dashboard (Power BI)

##  Project Overview
This project analyzes sales performance of ChocoRush, a chocolate retail business. The dashboard provides insights into total sales, loss sales, product distribution, and regional performance.

---

##  Objectives
- Analyze total sales and loss sales  
- Understand product distribution (boxes, categories)  
- Identify sales trends over time  
- Evaluate regional performance  
- Detect patterns affecting sales performance  

---

##  Tools & Technologies Used
- Power BI  
- Power Query Editor (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Excel / Dataset  

---

##  Key Features
- KPI Cards (Total Sales, Loss Sales, Total Boxes)  
- Sales trend analysis (time-based)  
- Region-wise performance  
- Product/category insights  
- Interactive filters  

---

##  Data Cleaning & Transformation
- Cleaned and structured raw data using Power Query Editor  
- Removed inconsistencies and handled missing values  
- Created calculated measures using DAX  

---

##  Key Insights
- Total boxes sold reached approximately 4M  
- Loss sales identified (~833), highlighting inefficiencies  
- Sales trends show fluctuations over time  
- Certain regions contribute more to total revenue
